A. MEMBUKA DENGAN MOZZILA FIREFOX
   1. buka file index.html dengan mozzila firefox
   2. Tunggu Hingga loading selesai
   3. klik proses dan tunggu (agak lama sekitar 30s)
   4. setelah selesai dan muncul table dan tombol download.
   5. klik download untuk download file.
   6. bila tidak bisa download gunakan cara kedua(B)

alternatif lain.
B. MEMBUKA DENGAN MENGAKTIFKAN WEB SERVER
   1. aktifkan webserver (xampp, lampp, atau sejenisnya)
   2. pindahkan semua file dan folder (dalam satu folder som) ke folder public server (htdocs)
   3. buka browser masukan alamat (localhost/som/index.html)
   4. Tunggu Hingga loading selesai
   5. klik proses dan tunggu (agak lama sekitar 30s)
   6. setelah selesai dan muncul table dan tombol download.
   7. klik download untuk download file.
